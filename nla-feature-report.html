🗺️ Platform User Journeys

The Fan Path

How supporters interact with the platform.

  • 1. DISCOVER

    • Browse stories and explore the Athlete Leaderboard.

  • 2. JOIN

    • Securely Sign Up or Login to unlock interactive features.

  • 3. ENGAGE

    • Support athletes via Likes, Comments, and Bookmarks.

The Athlete Path

The journey from athlete to storyteller.

  • 1. ONBOARD

    • Create account and verify identity via secure OTP.

  • 2. BRAND

    • Set up custom profile, social links, and S3 banners.

  • 3. PUBLISH

    • Upload multi-media stories and track fan engagement.

The Admin Path

Maintaining the platform integrity.

  • 1. OVERSEE

    • Login to Dashboard to view platform-wide statistics.

  • 2. MODERATE

    • Review pending stories and manage user permissions.

  • 3. MONITOR

    • Track system health, database health, and server latency.
